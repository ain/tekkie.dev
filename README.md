# tekkie.flashbit.net

Tech blog at http://tekkie.flashbit.net

## Usage

1. Install all dependencies

        $ bundle && npm i

2. Run Jekyll server

        $ jekyll s

3. Run BrowserSync

        $ npm run serve

4. When making a change, site will now automagically recompile and reload
